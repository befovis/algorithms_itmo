## Input / Output

| Input             | Output |
|--------------------|--------|
| 3 \n + 1 \n + 2 \n - | 1      |
| 4 \n + 3 \n + 4 \n - \n - | 3 4 |
| 5 \n + 5 \n + 6 \n - \n + 7 \n - | 5 6 |

## Input / Output

| Input                     | Output |
|----------------------------|--------|
| 5 \n push 1 \n push 2 \n max \n pop \n max | 2 1    |
| 3 \n push 3 \n max \n pop | 3       |
| 4 \n push 5 \n push 5 \n max \n pop | 5       |

## Ограничения по времени и памяти

## Задание
Определить, есть ли в массиве элемент, который встречается более чем в половине случаев. Программа должна вернуть 1, если такой элемент существует, и 0, если нет.

## Input / Output

| Input            | Output    |
|------------------|-----------|
| 6 \n 1 1 2 3 3 3 \n | 1 | 
| 5 \n 5 5 5 2 2 \n | 1 |
| 4 \n 1 1 1 1 \n | 1 |

## Задание
Нужно отсортировать много неотрицательных целых чисел.

## Input / Output

| Input              | Output      |
|--------------------|-------------|
| 3 3 \n 1 2 3 \n 4 5 6 | 90          |
| 4 4 \n 2 3 4 5 \n 1 2 3 4 | 62         |
| 2 2 \n 1 2 \n 3 4  | 14          |

## Ограничения по времени и памяти

- Ограничение по времени: 2 сек.
- Ограничение по памяти: 256 МБ.

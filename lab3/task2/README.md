## Задание
Умножить элементы двух массивов попарно, отсортировать результат, и вывести сумму каждого 10-го элемента.

## Input / Output

| Input    | Output |
|----------|--------|
| 5 \n | 1 2 3 5 4 |
| 6 \n | 1 2 3 4 5 6 |
| 4 \n | 1 2 3 4 |
